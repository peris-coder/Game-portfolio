# 🎮 Game Developer Portfolio (UI Design Project)

## 📌 Description
This project is a game developer portfolio website created as part of an academic assignment. It demonstrates my ability to design and build modern, responsive web interfaces using front-end technologies.

The portfolio represents a fictional game developer profile and showcases skills, projects, and contact features in a clean and visually appealing layout.



## 🎯 Objective
The goal of this project was to:
- Practice building a multi-page website
- Design a modern user interface (UI)
- Improve front-end development skills
- Present information in a structured and user-friendly way



## 🚀 Features
- About Me section (fictional developer profile)
- Skills section with visual progress bars
- Projects showcase page
- Contact page with form layout
- Responsive and modern design



## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  



## 💡 What I Learned
- Structuring and linking multiple web pages
- Designing clean and user-friendly interfaces
- Improving layout and styling using CSS
- Organizing project files effectively
